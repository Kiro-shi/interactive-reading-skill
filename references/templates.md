# Interactive Reading Templates

## Diagnostic Questions

Use up to 4 questions:

1. 你读这个材料最想解决什么实际问题？
2. 你现在对主题的熟悉度是 0-5 的哪一档？卡在哪里？
3. 你希望最后产出什么：复述、考试、写作、项目、决策、教学，还是行动方案？
4. 你有多少时间？希望先走最短路径，还是完整拆解？

## Learning Map

```markdown
# 学习地图

| 单元 | 优先级 | 依赖 | 要解决的问题 | 产出 | 检查点 |
|---|---|---|---|---|---|
| U1 | must | 无 |  |  |  |
| U2 | useful | U1 |  |  |  |
```

Priority labels:
- `must`: 不学会就无法达成目标。
- `useful`: 能明显提高理解或应用质量。
- `optional`: 有价值但不是当前目标必需。
- `skip`: 暂时不读，说明原因。

## Socratic Ladder

Use this progression:

1. 定义：你说的 X 是什么？不是什么？
2. 区分：X 和相邻概念 Y 的差别是什么？
3. 证据：作者凭什么这么说？
4. 机制：如果 X 成立，中间发生了什么？
5. 反例：什么时候 X 会失效？
6. 应用：把 X 用到你的场景，会改变哪一步？
7. 压缩：用一句话教给别人。

Adapt the ladder by material type:
- 认知/思维：多问失效条件、偏差、反例。
- 商业/投资：多问决策规则、风险、可验证信号。
- 写作/创作：多问人物、结构、情绪、场景、修改动作。
- 技术/方法：多问步骤、输入输出、错误处理、最小实践。
- 哲学/文学：多问价值判断、人生经验、语言表达、个人意义。
